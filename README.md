# Holi Festival - Festival of Colors

## Description
This project is a vibrant and interactive website dedicated to celebrating the Holi Festival, also known as the Festival of Colors. The website includes details about Holi, an event countdown, an image gallery, event schedules, and a contact section.

## Features
- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Event Countdown:** Displays a real-time countdown to the Holi 2025 celebration.
- **Gallery Section:** Showcases images of past Holi celebrations with filtering options.
- **Event Schedule:** Lists upcoming Holi-related events with descriptions and registration links.
- **Contact Form:** Allows users to send messages for inquiries and event participation.
- **Social Media Links:** Connects users to social platforms for updates and engagement.

## Technologies Used
- **HTML5** - Structure of the website
- **CSS3** - Styling and animations
- **JavaScript** - Interactive elements
- **FontAwesome** - Icons for enhanced UI
- **Google Fonts** - Custom fonts for better aesthetics

## Installation & Usage
1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.
3. Ensure the linked CSS (`style.css`) and JavaScript (`script.js`) files are in the same directory.
4. To make modifications, edit the respective HTML, CSS, and JS files.

## Folder Structure
```
- index.html  (Main HTML file)
- style.css   (Stylesheet for the website)
- script.js   (JavaScript for interactivity)
- images/     (Contains all images used in the website)
- audio/      (Music files for the Holi experience)
```
